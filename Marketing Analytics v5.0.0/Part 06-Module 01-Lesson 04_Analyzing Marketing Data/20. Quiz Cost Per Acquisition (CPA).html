20. Quiz: Cost Per Acquisition (CPA)

Which one would you use if you were asked to show how much did it cost to get a lead?

Select all that apply.
SOLUTION:
  • Cost Per Lead

Which one would you use if you were asked to show how much did it cost to get a paying customer?
SOLUTION: Customer Acquisition Cost

Please download the BAND CPA Quiz CSV file located under the Resources on the left side of the page. You can use the file to calculate the appropriate metrics to complete the following two quiz questions.

Which of the following three months had the highest Cost Per Acquisition?
SOLUTION: January

QUIZ QUESTION::

Match the Cost Per Acquisition for each month based on the BAND CPA Quiz file.

ANSWER CHOICES:



Month

Cost Per Acquisition

January

February

March
SOLUTION:

Month

Cost Per Acquisition

February

January

March

Recap:

  • Cost Per Acquisition (CPA) = (Marketing and Sales Cost)/ number of new leads customers

  • CPA is referring to marketing + sales costs (overhead, salaries) in the numerator and includes only leads (non-paying customer) in the denominator.

  • Here “acquisition” refers to a non-paying customer.

Three Views Best Practice

Criteria Meet Specification

Did the student create three views in keeping with the best practice, demonstrate that it had already been done, or show where and explain how to do it?

Your screenshot should clearly show all three views created in the same property or show where this would be done by providing clear, ordered steps to follow. Providing a rationale for the creation of the views is also important.

Did the student document somewhere on the page whether they are using the Google Merchandise Store Demo Account, or their own business?

The student has clearly documented whether they are using the Google Merchandise Store Demo Account or their own business for this project.

Filtering Out Internal Traffic

Criteria Meet Specification

Did the student show evidence of building a filter to exclude internal traffic from reporting or show that one was already in place?

The filter captures internal traffic, even if it comes in from various locations, IP addresses, etc.

Switching Displays to Try and Answer Questions

Criteria Meet Specification

Does the student provide additional notes or questions where relevant to demonstrate an understanding of the report types?

Insights and findings that have been uncovered from each reporting view have been documented and explained. If specific questions have been asked in the template, they are answered by the student.

Creating Segments

Criteria Meet Specification

Did the student show evidence of creating an audience-based segment?

Audience segments include screenshots of each: Characteristic, Geography, and User Behavior.

Has the student provided a written explanation of each segment and the results they have found?

For each segment, clear and detailed notes should explain what type of data each segment is providing, and what results/insights can be found from the segment.

05. Tableau Public / Tableau Desktop

Save Every 5 Minutes

Tableau Public can be a little bit buggy. It is highly recommended that you save as frequently as possible so that you do not lose your work.

If you do run into an issue saving your work please follow these instructions:

At the bottom of the tableau editor window that you are working in there should be a drop down that has your name. Click it open and select sign out. DO NOT CLOSE THE WINDOW. All you are doing is signing out down there. Then go up to the top of the window and click on the square that looks like an old zip disk that is the save icon. It should prompt the opening of the sign in window. Log back in there and it should save your work.

If you are unable to save to Tableau Public

Please include screenshots of your visualizations inside your PDF report. This is not preferred but we realize this may be necessary and is an acceptable option if you need to use it.

If you choose to use screenshots you should include at least one screenshot of your filters being used (a before and after picture of the visualization).

Tableau Desktop

If you get an error around needing an extract of your data while using Tableau Desktop. Navigate to the Data Source tab at the bottom of the screen and then click Create Extract at the top right of the Data Source page. If you are unable to do this please reach out on Study Groups or Knowledge so a mentor or fellow student can help further.

04. Exploring the Data III

Visuals

To really understand any of the variables well, we should probably just create visuals. A visual is so much more useful for explaining what is going on when we have variables that can take on lots of different values. The main way we might look at the categorical variables is using bar charts, and the main method for looking at quantitative variables is using histograms.

You will need to use software to create your own visuals, but in this section, we walk through some of the visuals I created from this dataset. Below are histograms for many of our variables. Some of them are not very informative because I have not done anything with outliers (something you will want to do in the second part of this project).

The amount of sleep histogram is all one bar at 0. What is going on here?
SOLUTION: There was an outlier in the dataset that scrunched most of the data into one bin.

Helpful Hint: Above are two histograms of the same sleep data. However, one of the histograms has individuals who answered more than 12 hours removed. Clearly one of these is more informative than the other. Make sure you build plots that are informative when you are creating your presentation.

05. Quiz: Ecommerce Configuration

(True or False) Standard ecommerce tracking can be configured in GTM, but enhanced ecommerce requires inline tracking code to be placed on your pages manually.
SOLUTION: False

Which of these is not an example of a product-scope custom dimension you could use in your ecommerce implementation?

SOLUTION: Campaign

(True or False) Implementing both standard and enhanced ecommerce require only the toggle of button for each in the view setting where you wish to have it activated.
SOLUTION: False

Which of the following are required fields in standard ecommerce tracking?
(Select all that apply.)
SOLUTION:
  • Transaction ID
  • Product ID

SOLUTION: False

What can allow you to identify drop-off points in the shopping process so you can retarget those shoppers that were lost?
SOLUTION: Both a. and b.
